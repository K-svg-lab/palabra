## üé® **Screenshot Design Guidelines**

### **Layout & Composition**

**Safe Zones:**
- **Top:** 100px (avoid status bar area)
- **Bottom:** 100px (avoid home indicator)
- **Sides:** 40px (comfortable margins)

**Text Overlays (Optional):**
- Keep overlays minimal (App Store shows captions below)
- Use if adding context is critical
- Font: SF Pro (iOS) or Roboto (Android)
- Size: 24-32px (readable but not dominant)
- Color: Semi-transparent dark or light overlay

### **Visual Style**

**Consistency:**
- Same time in status bar (9:41 for iOS tradition)
- Same battery level (100% or hide)
- Same network signal (full bars)
- Clean status bar (no notifications)

**Data:**
- Use realistic but impressive data
  - Streak: 7-14 days (believable but motivating)
  - Vocabulary: 200-500 words (established user)
  - Accuracy: 75-85% (good but not perfect)
- Avoid perfect scores (looks fake)

**Dark Mode:**
- Show 3-4 screenshots in light mode
- Show 2-3 screenshots in dark mode
- Demonstrates dual theme support
