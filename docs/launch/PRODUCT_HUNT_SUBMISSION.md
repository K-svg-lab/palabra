### **"How is this different from Duolingo?"**
```
Great question! Duolingo focuses on full language learning (grammar, reading, speaking), while Palabra specializes in vocabulary acquisition. We're laser-focused on helping you build a strong vocabulary foundation through:

1. **Varied practice** - 5 different review methods vs repetitive flashcards
2. **AI examples** - Contextual examples that match YOUR level
3. **Smart scheduling** - Research-backed spaced repetition algorithm
4. **Progress tracking** - Beautiful visualizations of your growth

Think of it as: Duolingo = full language course, Palabra = vocabulary specialist.

You can use both together! Many users do Duolingo lessons and add new words to Palabra for deeper practice.
```

```
Different tools for different needs! Anki is incredibly powerful and flexible, but it has a steep learning curve. Palabra is designed for ease-of-use:

**Anki**: Powerful, customizable, complex setup
**Palabra**: Easy, beautiful, works out of the box

If you're a power user who loves customization â†’ Anki
If you want something that "just works" â†’ Palabra

We focus on Spanish vocabulary specifically, with AI-generated examples and level-appropriate content. Anki is general-purpose.
```


### **"How does the AI work?"**
```
We use OpenAI's GPT-3.5-turbo to generate contextual examples tailored to your proficiency level (A1-C2). The AI:

4. Adapts complexity to match your ability
```
